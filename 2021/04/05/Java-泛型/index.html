发表于 2021-04-05 分类于 java

概述

泛型，即”参数化类型“，将类型有原来的具体类型参数化，类似于方法中的变量参数，此时类型也定义成参数形式（也可以称为类型形参），在使用/调用时传入具体的类型（类型实参）。泛型的使用有泛型类，泛型接口，泛型方法
